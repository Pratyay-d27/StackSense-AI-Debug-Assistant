## 🧠 What is StackSense?

**StackSense AI** is a next-generation VS Code extension that brings **context-aware AI debugging** directly into your workflow.

⚡ No need to paste full code  
⚡ Just ask → AI already understands your file  
⚡ Get explanations, fixes, and optimization instantly
